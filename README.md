# 📊 HR Analytics Dashboard | Microsoft Excel

An interactive **HR Analytics Dashboard** built entirely in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Excel formulas to transform raw HR data into actionable business insights.

---

# 📌 Project Overview

This project demonstrates how Microsoft Excel can be used to build professional and interactive HR dashboards without any external BI tools.

The dashboards provide key insights into employee demographics, performance, salaries, promotions, training, attendance, and turnover, enabling HR departments to make data-driven decisions.

---

# 📊 Dashboard 1 – Employee Overview

## Key Performance Indicators (KPIs)

- 👥 Total Employees
- 🎂 Average Employee Age
- 💰 Average Salary
- ⭐ Average Performance Rating
- 🏆 Highest Performance Rating

## Visualizations

- Employee count by job title
- Average performance by job title
- Employee distribution by age group
- Promotion eligibility rate
- Gender distribution

## Interactive Filters

- Department
- Gender
